이미지 관련 태그

src 속성에 경로를 설정하여 이미지 파일을 불러온다.


자주 사용하는 속성들

alt 속성

사진의 경로가 잘못되거나 이미지를 제대로 표현할 수 없는 경우 대체 텍스트 용도로 사용


width, height 속성

        사진의 높이와 너비 지정
        고정된 단위의 크기로도 지정 가능, 가변크기 단위의 크기로도 지정 가능

고정 크기 단위: 화면 사이즈가 줄어도 크기가 변하지 X

가변 크기 단위: 화면 사이즈 혹은 기준에 따라 크기 변경
